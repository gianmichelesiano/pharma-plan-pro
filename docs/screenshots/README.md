## Criteri Per Fare Gli Screenshot

- Usa sempre la stessa lingua dell'interfaccia nel documento finale.
- Mantieni una dimensione finestra coerente tra una schermata e l'altra.
- Evita dati sensibili reali, se possibile.
- Mostra i casi piu utili da spiegare, non schermate vuote.
- Se una funzione e riservata all'amministratore, cattura la schermata con un account admin.
